#  >_ Ascii Studio

A lightweight, high-performance web application that transforms your live camera feed into real-time ASCII art. Built with a stunning **Y2K Brutalist** aesthetic, this project requires zero backend, zero API keys, and runs 100% locally in your browser.

## ✨ Features

- **Real-time ASCII Rendering:** Optimized string-based canvas rendering for butter-smooth 60fps performance on any device.
- **Y2K Brutalist Landing Page:** An incredibly highly-stylized animated landing page featuring neon marquees and 8-bit Pacman pixel art animations.
- **Dual Color Themes:** Dynamically separate yourself from the background using real-time native luminance thresholding. 
- **Boomerang Video Capture:** Record, generate, and instantly download a hyper-lapse looping ASCII video straight from the browser (exported locally as `.mp4`).
- **Snapshot Gallery:** Snap high-quality ASCII photos and manage them in your local browser gallery, saving your data directly to `localStorage`.
- **100% Client-Side Privacy:** No external APIs, no backend servers, and no tracking. Your webcam feed never leaves your computer.

## 🚀 Quick Start

**Prerequisites:** Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone <https://github.com/AR10129/asciiStudio>
   cd asciiStudio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
   *The application will automatically open in your default browser at `http://localhost:5173`.*
